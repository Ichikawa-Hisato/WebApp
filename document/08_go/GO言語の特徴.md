# GO言語のメリット
* 初心者でも理解しやすい
* 処理の実行速度が速い
* 安全性が高い
* 並行処理・並列処理が得意
* ライブラリが豊富

# GO言語のデメリット
* 情報量が少ない
* Genericsがない（2021年6月時点）
* コードの継承がない
* 例外処理がない
    * シンプルなコードを目指すため必要ないと判断されている。

# GO言語でできること
* Webサーバー構築
* アプリケーション開発
* コマンドラインインターフェース（CLI）ツール作成

# Go言語の学習方法
* 書籍を読んで学習する
* 公式サイトで学習する
    * 公式サイトのチュートリアルを使う方法がある。
    * チュートリアルは、Go言語について体系的に学べるプログラムになっている。
    * 公式サイトURL：https://go.dev/
        * 日本語訳ページ
            * https://go-tour-jp.appspot.com/welcome/1
* スクールに通って学習する
