# ラムダ式の概要
「ラムダ式(lambda expressions)」は、簡易的な関数オブジェクトをその場で定義するための機能である。  
この機能によって、「高階関数(関数を引数もしくは戻り値とする関数)」をより使いやすくできる。  
※高階関数とは「関数を引数や戻り値とする関数」のこと

* 高階関数を使いたい場面
    * 一言で表すと、以下のような関数を作りたい場合
        * 状態を持った関数（クロージャー）を作りたい
        * カリー化したい
        * 定型的な関数を楽に作りたい

* クロージャーとは？
    * 状態を保持する関数のこと
        * 状態とは、ほとんどのプログラミング言語において、変数のこと。変数を状態として、関数内に閉じ込めておくから、クロージャ。
        * 例)
            * 問：呼び出すたびに、1,2,3,...を返すような関数f()を定義せよ
                ```
                f(); // 1
                f(); // 2
                f(); // 3
                ```
            * これを引数を取らずにこれを実現できるのがクロージャ

* カリー化とは？
    * 「複数の引数をもつ関数」を分割するということ
    * カリー化はプログラミング上のテクニックというよりは、理論的な面での扱いやすさ、という側面が大きい
